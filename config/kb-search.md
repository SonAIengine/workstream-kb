# 지식 베이스 검색 (Knowledge Base Search)

$ARGUMENTS 키워드로 지식 베이스를 검색합니다.

## 실행 순서

### 1단계: 인덱스 로드
- `~/workstream-kb/index.json` 파일을 읽어 검색 인덱스를 로드

### 2단계: 검색 실행
- 키워드: `$ARGUMENTS`
- 검색 대상 필드: title, date, type
- 대소문자 무시, 부분 일치 허용
- 키워드가 여러 단어인 경우 OR 검색 (하나라도 매치되면 포함)

### 3단계: 결과 출력

아래 형식으로 출력:

## 검색 결과: "$ARGUMENTS"

| # | 제목 | 날짜 | 메시지 수 |
|---|------|------|----------|
| 1 | ... | ... | ... |

총 N건 검색됨

검색 결과가 없으면:
> 검색 결과가 없습니다. 다른 키워드로 시도해보세요.

### 4단계: 상세 보기
검색 결과가 있으면 사용자에게 번호를 선택하여 상세 내용을 볼 수 있다고 안내.
선택 시 해당 `path`의 마크다운 파일을 읽어 전체 내용을 표시.
